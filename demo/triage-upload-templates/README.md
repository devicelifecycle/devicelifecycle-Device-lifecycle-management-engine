Triage upload demo files for client walkthroughs.

Files in this folder:
- `triage-client-demo.csv`: sample intake file with six devices, IMEIs, grades, and notes.

Recommended demo flow:
1. Create or open a draft order with matching devices.
2. Upload `triage-client-demo.csv` in COE Triage.
3. If needed, link the order number in the triage upload panel to rematch the rows.
4. Review the preview counts for order-matched, condition mismatches, and unmatched rows.
5. Import matched rows into the triage queue.
